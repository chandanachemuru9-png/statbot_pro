# 📊 StatBot Pro — AI CSV Data Analyst Agent

> "Turn your data into insights — just by asking!"

## 🎯 About
StatBot Pro is an AI-powered web application that allows anyone 
to upload a CSV or Excel file and analyze their data by asking 
questions in plain English — no coding or technical knowledge needed.

## 😕 Problem
Most people cannot analyze data because they don't know 
coding or Excel formulas. Data goes unused and insights are lost.

## ✅ Solution
Upload your file → Ask a question → Get instant AI answer + charts!

## 🔄 How It Works
1. Upload your CSV or Excel file
2. Ask questions in plain English
3. Get instant AI answers
4. Generate beautiful charts

## ✨ Features
- 💬 Natural language data querying
- 📊 5 Chart types (Bar, Line, Pie, Scatter, Histogram)
- 🕓 Chat history saved
- 📈 Chart history saved
- 📥 Download answers as text
- 📁 CSV and Excel file support
- 🌙 Dark theme UI
- 🔒 Secure API key management
